## Setup dev environment

```bash
poetry env use python3.12 # В случае ошибки указать полный путь до интерпретатора вместо python3.12
poetry install
poetry run pre-commit install
```
## The entry point to the project.
src.gui.__main__
or
```bash
poetry run ssk-gui
```
